# TheKitchen
Recipe app built for my mom but you can check it out too. 

MERN stack application. Create an account and add pre listed recipes to your favorites and add them to a custom folder for each user. View favorites by selecting your username in the top right and select your profile. Inside the profile, you can add your favorite recipes to a folder you create. Users can post reviews on the recipes.

Deployed on heroku at https://jorecipe.herokuapp.com/
